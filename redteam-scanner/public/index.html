Comprehensive analysis of DOM structure, hidden elements, and dynamic content

Sensitive Data Detection

Vulnerability Assessment

Overlay Removal

Real-time Monitoring

Monitor network requests, WebSocket connections, and API calls in real-time

How to Use

1

Install the Bookmarklet

Drag the bookmarklet to your bookmarks bar for easy access on any website

2

Navigate to Target Site

Visit the website you want to scan for security vulnerabilities

3

Run the Scanner

Click the bookmarklet or use keyboard shortcuts to start the scan

4

Review Results

Analyze the comprehensive security report with categorized findings

Keyboard Shortcuts

Start Scan: Ctrl+Shift+S
Remove Overlays: Ctrl+Shift+R
View Results: Ctrl+Shift+V
Close Results: Escape

Console Commands

Open the browser console and use these commands:

RedTeamScan.start()

Start a comprehensive security scan

RedTeamScan.removeOverlays()

Remove overlays, paywalls, and blur effects

RedTeamScan.viewResults()

View the last scan results

RedTeamScan.exportResults()

Export scan results to console
